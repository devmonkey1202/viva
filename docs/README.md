## 문서 운영 원칙

- 문서와 코드가 충돌하면 그대로 두지 않는다.
- 같은 내용을 여러 파일에 반복하지 않는다.
- 문서는 실행 기준을 고정하는 역할을 한다.
- 기록은 남기되 역할이 겹치면 통합한다.
