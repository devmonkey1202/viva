## 1. 필수 제출물

## 2. 저장소 제출 규칙

- 저장소는 반드시 public 이어야 한다.
- API Key, 토큰, 시크릿은 노출되면 안 된다.
- 저장소에는 AI와 협업한 문서와 지침서를 포함하는 것이 유리하다.
- README 첫 화면에서 문제와 솔루션이 바로 보여야 한다.

- 기술적 완성도: 최소 기능이 끊김 없이 데모되어야 한다.
- AI 활용 능력 및 효율성: 왜 AI가 필요한지, 어떻게 효율화했는지 보여야 한다.
- 기획력 및 실무 접합성: 실제 교사 pain과 운영자 pain이 설명돼야 한다.
- 창의성: LMS/튜터/탐지기와 다른 관점 전환이 보여야 한다.
